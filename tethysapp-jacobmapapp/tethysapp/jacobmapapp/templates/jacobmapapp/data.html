Data Services

The data services used in this app are provided by ArcGIS, and the ArcGIS API using JavaScript. This app integrates a single 2D map for air quality prediction. Two data sets are used in this app. One is a DEM of Utah County in Utah. This DEM was obtained from the Utah AGRC website under USGS DEM and NED Indices. The second data set is a shapefile that contains vector data representing roads in Utah. This data set was obtained from the Utah AGRC website under Roads and Highway System. The data is hosted on ArcGIS Map Services and can be accessed here. The DEM data set is included as a map service while the Shapefile is included as a feature service.
